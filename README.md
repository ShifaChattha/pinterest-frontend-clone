# Pinterest Front-End Clone

A responsive web application that replicates the Pinterest masonry layout. This project demonstrates my ability to work with dynamic data fetching and responsive CSS design.

## Live Demo
[View Live Project](https://shifachattha.github.io/pinterest-frontend-clone/index.html)

## Tech Stack
HTML5/CSS3: Used for the masonry grid and hover effects.
JavaScript (Vanilla):Handles the logic for fetching data.
XML: Used as the data source to populate the image cards dynamically.
